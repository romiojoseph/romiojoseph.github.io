A simple tool to search Bluesky posts, users, and tags. Just type what you need, and it’ll open a new tab with the right search format. This tool is based on tips from the Bluesky blog, which says: “Over time, we'll make these features easier to use directly within the Bluesky app. Think of these as advanced features!”

JavaScript made by Mistral AI. Built for regular folks who don’t know or want to deal with search tricks like from:, to:, mentions:, hashtags, or date filters. You can even paste a URL into the Bluesky search to find posts that shared it.
