Bluesky Post Analyzer

Analyze Bluesky posts with detailed stats - track likes, reposts, replies, and quotes. View engagement timelines, user interactions, and performance metrics.

Published on 17 Feb, 2025 Updated on 13 Apr, 2025 in Bluesky

Enter a Bluesky post URL to analyze posts with detailed stats — track likes, reposts, replies, and quotes. View engagement timelines, user interactions, and performance metrics.

No authentication is needed. Bluesky is an open platform. But respect authors' privacy when using this tool.

Disclaimer

This project is part of my ongoing exploration of the Bluesky API. While I’ve built other tools for user stats, this one focuses only on post stats. Note: the repost timestamp isn’t available from the getRepostedBy API, so that data won’t show in the chart.

You can track how any Bluesky post performs over time. Replies and quotes show up in their own tables. You can sort and search through the rows. You’ll also see basic stats of users who liked or reposted — like followers, following, total posts, labels, and join date. This helps check if they’re bots, spam accounts, or real users.

If there are many interactions, loading all the data might take a while. These stats are helpful for folks using Bluesky for marketing. Regular users can also get a quick idea of how their posts are doing. This website runs fully in your browser using the Bluesky AppView API. Nothing is sent anywhere. No analytics or trackers.
