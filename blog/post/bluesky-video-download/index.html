Download Bluesky Video

Easily download Bluesky videos in available resolutions or get the playlist URL to stream. Respect authors' privacy and give proper credit when using this tool.

Published on 23 Jan, 2025 Updated on 13 Apr, 2025 in Bluesky

Bluesky is open. If you don’t want to download the video, you can stream it using VLC. Copy the playlist URL below, open VLC, go to Media > Stream > Network, and paste the link.

If you choose to download, please credit the author and respect any privacy notes in their profile or post.

###### Disclaimer This tool is meant for personal use. Since Bluesky is public, anyone can access posts—even without an account.

Videos are streamed using M3U8 playlist files. These are simple text files listing media sources like resolution options. Each Bluesky video post has a playlist URL. Clicking on the URL will download the M3U8 file. You can paste this URL into VLC or download a specific resolution from it.

The tool doesn’t store or track anything. How you use it is up to you.

Extra notes

  • DIDs (Decentralized Identifiers): Permanent account IDs for decentralized identity.
  • CIDs (Content Identifiers): Hashes that track specific versions of records.
