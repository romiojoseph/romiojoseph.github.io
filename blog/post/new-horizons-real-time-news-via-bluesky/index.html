I built this with the intention of staying in the loop on what is happening in the world, without targeted notifications or rage bait algorithms.

This is something I built for my own use, which helps me spot issues and improve it continuously. Bluesky has become my primary source of information because verification actually means something here, unlike paid verification systems elsewhere.

You can install it as a PWA. You can also browse threads from Hacker News.

Stream news with a link

This website runs entirely in your browser via the Bluesky AppView API, with no analytics or trackers. All JavaScript was built with the help of LLMs

As more companies, institutions, and organisations join Bluesky, it becomes a one-stop source for real-time updates. This project aggregates posts from Bluesky media profiles, tech companies, journalists, and organizations, and presents them in a news media format.

Nothing is stored. Posts display as they arrive and cycle out as new ones come in. Each post comes with its verification records: DID, AT URI, and a direct link to the original post on Bluesky. All content originates from Bluesky and remains the property of the respective account holders. This project does not claim ownership over any of it. It is part of my ongoing exploration of the Bluesky API.

Free & Open-Source Bluesky Tools A collection of free and open-source Bluesky tools, with every JavaScript created using AI, aka vibe coding. No ads, no paywalls and no trackers.
